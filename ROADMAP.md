# Gesahan News Pro

Premium WordPress News Theme

---

# Visi Proyek

Membangun tema WordPress Portal Berita Premium yang:

- Modern
- Ringan
- Cepat
- SEO Friendly
- Mobile First
- Responsif
- Mudah dikustomisasi
- Menggunakan WordPress Coding Standards
- Tanpa Bootstrap
- Tanpa Tailwind
- Menggunakan CSS dan JavaScript murni

Target pengguna:

- Portal Berita Nasional
- Portal Berita Daerah
- Media Pemerintah
- Media Kampus
- Media Perusahaan
- Majalah Online
